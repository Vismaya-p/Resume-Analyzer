# Smart Resume Analyzer

A Python-based tool that analyzes resumes and compares them with job descriptions using Natural Language Processing (NLP).

##  Features
- Extracts text from PDF resumes
- Compares resume with job description
- Calculates similarity score
- Helps identify job-fit percentage

##  Tech Stack
- Python
- Scikit-learn
- PyPDF2

## How to Run

```bash
pip install -r requirements.txt
python main.py
